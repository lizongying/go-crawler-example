# go-crawler-example

[go-crawler-example](https://github.com/lizongying/go-crawler-example)

## Usage

* log level: DEBUG/INFO/WARN/ERROR

example

```shell
git clone github.com/lizongying/go-crawler-example
```

build

```shell
make
```

run

```shell
./releases/youtubeSpider -c example.yml
```
